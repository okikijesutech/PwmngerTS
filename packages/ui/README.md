# @pwmnger/ui

Shared React component library for PwmngerTS.

## Components
- **Button:** Highly customizable, accessible button components.
- **Input:** Secure input fields with validation.
- **Card/Glass:** Premium glassmorphism layout components.
- **Toast:** Alert and notification system.

## Design System
The UI package follows a "Premium Glassmorphism" aesthetic with high-contrast typography and subtle micro-animations.
