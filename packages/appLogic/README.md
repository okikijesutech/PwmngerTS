# @pwmnger/app-logic

High-level business logic for PwmngerTS vault and authentication flows.

## Features
- **Vault Management:** Unlock, Sync, and Entry CRUD operations.
- **Authentication:** Login and Register flows with 2FA enforcement.
- **Orchestration:** Coordinates between `@pwmnger/crypto`, `@pwmnger/storage`, and the Backend API.

## Usage
This package provides a unified API for all PwmngerTS clients (Web, Extension, Mobile).
